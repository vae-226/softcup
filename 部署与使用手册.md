# 部署与使用手册

## 1. 产品简介

灵境导游是一套面向景区场景的 AI 数字人智能导览系统，包含游客交互端和管理后台两部分。系统面向比赛场景提供可演示的原型版本，重点展示多模态交互、知识问答、路线推荐和运营分析能力。

## 2. 运行环境

本版本为纯前端静态原型，无需安装数据库或复杂依赖。

- 操作系统：Windows / macOS / Linux
- 浏览器：Chrome、Edge、Firefox 最新版
- Python：可选，用于启动本地静态服务

## 3. 部署步骤

### 方式一：直接运行

1. 进入项目目录
2. 双击打开 `index.html`
3. 在浏览器中查看系统演示

### 方式二：HTTP 服务运行

1. 打开终端并切换到项目目录
2. 执行以下命令：

```powershell
python -m http.server 8000
```

3. 浏览器访问：

[http://localhost:8000](http://localhost:8000)

### 方式三：启用千问 Qwen 真实 AI 接口

1. 获取阿里云百炼 DashScope API Key
2. 在 PowerShell 中进入项目目录
3. 执行：

```powershell
$env:DASHSCOPE_API_KEY="你的DashScope API Key"
python server.py
```

4. 浏览器访问：

[http://127.0.0.1:8000](http://127.0.0.1:8000)

说明：前端会优先调用 `POST /api/qwen/chat`。如果没有配置 Key 或网络异常，系统自动降级为本地知识库回答，保证演示不中断。

## 4. 功能使用说明

### 4.1 游客交互端

游客端提供以下能力：

- 文本输入提问
- 语音输入模拟
- 数字人状态展示
- 路线推荐查看
- 景区知识问答

建议演示流程：

1. 进入“游客交互端”
2. 点击“历史路线”“夜游时间”“拍照推荐”快捷按钮
3. 输入“我带着孩子来玩，推荐一条亲子路线吧”
4. 观察数字人表情、聊天回复与路线推荐模块联动

### 4.2 管理后台

管理后台包括：

- 今日/本周服务数据
- 游客满意度趋势
- 热门问答统计
- 游客感受度报告
- 知识库管理表格
- 数字人形象配置

建议演示流程：

1. 切换到“管理后台”
2. 查看 KPI 指标和情绪趋势图
3. 演示“模拟上传资料”
4. 切换数字人形象配置并保存

## 5. 当前版本说明

本版本用于初赛展示，数据为示范景区模拟资料，AI 能力为前端本地逻辑模拟。其优势是：

- 可立即运行，适合答辩和视频录制
- 页面完整，覆盖比赛主要评分点
- 后续可快速接入真实模型与后端服务

## 6. 升级建议

为满足正式落地或决赛需求，建议补充以下模块：

1. 接入真实 ASR/TTS 服务
2. 接入多模态大模型 API
3. 构建向量知识库
4. 对接真实数字人驱动引擎
5. 增加账号体系、日志系统与数据库持久化
6. 增加 GPS/蓝牙定位与弱网应对机制
